# Electron Vue Vite App

A clean Electron desktop application scaffold using Vue 3 and Vite.

## Scripts

- `npm run dev` starts Vite and Electron together for local development.
- `npm run build` builds the Vue renderer into `dist`.
- `npm run pack` creates an unpacked desktop app in `release`.
- `npm run dist:win` builds Windows NSIS and portable packages.
- `npm run dist:mac` builds macOS DMG and ZIP packages.

## Project Structure

- `electron/main.cjs` creates the desktop window and loads the Vite app.
- `electron/preload.cjs` exposes a small safe bridge to the Vue renderer.
- `src/` contains the Vue application shell.
- `dist/` is generated by Vite.
- `release/` is generated by electron-builder.

## Packaging Notes

Windows packages can be built on Windows. macOS packages are configured here, but signed and notarized macOS releases should be produced on macOS with Apple Developer credentials.
